VRShopping mobile application.
An interactive mobile VR application to experience a Virtual shopping experience from anywhere.
Steps:
1. Download the application in android.
2. Go to 2D shopping site https://ullaged-multiplex.000webhostapp.com/index.php and register.
3. Using the register login in VRS application.
4. Enjoy the shopping experience.

Using Google Cardboard, experience 360 degree view of shopping center.

To walk, turn your head downwards.(Turn the angle of pointer downward to walk Virtually)

To interact, pointer in VRS will helps to interact with object in 3D shopping.

Look the object in Virtual environment for few milliseconds to get the details of product on screen which is placed in Virtual environment.
